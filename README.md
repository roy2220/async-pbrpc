# async-pbrpc

Lightweight RPC framework based on AsyncIO and Protocol Buffers

## Requirements

- python >= 3.6.0
- protobuf-compiler >= 3.3.1

## Installation

```
python3 -m pip install -U --process-dependency-links git+git://github.com/roy2220/async-pbrpc.git
```
